# DevMedia

A escola de programação com a melhor didática do Brasil.
25 anos desenvolvendo um método próprio, baseado em como as pessoas
realmente aprendem.

## Sobre a plataforma

A DevMedia oferece uma metodologia construída em 25 anos de ensino,
com cada etapa pensada para o aluno avançar com confiança, sem pular
etapas e sem se perder.

## Tecnologias abordadas

- Front-End: HTML, CSS, JavaScript, React e mais
- Back-End: Node.js, Express, APIs e mais
- Mobile: React Native e publicação nas lojas
- Banco de Dados: SQL, MySQL e modelagem
- Deploy e Versionamento: GitHub, Vercel e CI/CD
- IA e Automação: ChatGPT, Claude, Copilot, n8n e mais

## Este perfil

Os repositórios desta conta reúnem projetos desenvolvidos durante
os cursos da DevMedia, criados para fins de estudo e prática.
